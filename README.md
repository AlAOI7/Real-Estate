# موقع المقاولات والبناء والتشطيب

هذا المشروع هو نظام متكامل لإدارة خدمات المقاولات والبناء والتشطيب وأعمال الأحواش، يحتوي على:

- لوحة تحكم للإدارة
- نظام تسجيل دخول آمن
- إدارة الفئات والمشاريع والصور
- قاعدة بيانات MySQL
- مبني بلغة PHP

---
